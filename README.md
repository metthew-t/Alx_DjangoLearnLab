# LibraryProject

Django project created for ALX Django learning lab.

## Setup
1. `pip install django`
2. `django-admin startproject LibraryProject`
3. `cd LibraryProject`
4. `python manage.py runserver`

## Structure
- `manage.py`: Django management script
- `LibraryProject/settings.py`: Project configuration
- `LibraryProject/urls.py`: URL configuration
